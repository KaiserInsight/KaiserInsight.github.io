Markdown 基础教程

Markdown 是一种简洁的标记语言，用于书写结构化文本。以下内容帮助你从基础到进阶掌握常见语法。

标题

使用一个或多个 # 表示标题层级：

# 一级标题
## 二级标题
### 三级标题
#### 四级标题

段落与换行

输入连续文本即可生成段落。要强制换行，可在行尾添加两个空格：

第一行␣␣
第二行

也可以通过空行分隔段落。

强调与加粗

使用星号或下划线表示强调：

*斜体* 或 _斜体_
**加粗** 或 __加粗__
***加粗斜体***

列表

无序列表：

- 项目一
- 项目二
  - 子项目
  - 另一子项目
- 项目三

有序列表：

1. 第一项
2. 第二项
3. 第三项

任务列表：

- [x] 已完成任务
- [ ] 待办任务

链接与图片

链接语法：

[点击这里](https://example.com)

图片语法：

![图片说明](https://example.com/image.png)

也可以添加标题属性：

![图片说明](https://example.com/image.png "悬浮标题")

代码与高亮

行内代码使用一个反引号：

这是 `const x = 1` 的示例。

多行代码块使用三个反引号，并可指定语言：

```python
print("Hello, Markdown")


### 引用

使用 `>` 表示引用：

```markdown
> 这是一个引用段落。
> 可以换行并保持引用格式。

表格

表格用于展示结构化数据：

| 名称 | 类型 | 说明 |
| --- | --- | --- |
| Markdown | 标记语言 | 适用于文档和博客 |
| HTML | 标记语言 | 更强的表现力 |

效果如下：

名称类型说明
Markdown标记语言适用于文档和博客
HTML标记语言更强的表现力

分隔线

使用三条或更多短横线表示分隔线：

---

代码引用与嵌套

你可以在列表中嵌套代码块：

1. 步骤一
2. 步骤二

   ```bash
   echo "Hello"


### 斜体、粗体与删除线组合

```markdown
**加粗**
*斜体*
~~删除线~~
***加粗斜体***

链接复用与自动链接

引用式链接：

这是一个[示例链接][1]。

[1]: https://example.com

自动链接：

<https://example.com>

摘要

Markdown 使用简单的标记符号来表达结构，适合写作说明文档、教程、博客和笔记。掌握标题、列表、链接、图片、代码和表格，就能应对大部分写作需求。
