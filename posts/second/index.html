Python 基础教程

1. Hello World

print("Hello, World!")

输出：

Hello, World!

2. 注释

单行注释

# 这是一个注释
print("Python")

多行注释

"""
这是多行注释
可以写多行内容
"""

3. 变量与数据类型

Python 不需要声明变量类型。

name = "Tom"
age = 18
height = 1.75
is_student = True

常见数据类型：

类型示例
字符串 str"hello"
整数 int123
浮点数 float3.14
布尔 boolTrue
列表 list[1,2,3]
元组 tuple(1,2,3)
字典 dict{"a":1}
集合 set{1,2,3}

查看类型：

x = 10
print(type(x))

4. 输入输出

输出

print("你好")

多个变量：

name = "Tom"
age = 20

print(name, age)

格式化输出：

name = "Tom"
age = 20

print(f"{name}今年{age}岁")

输入

name = input("请输入名字:")
print(name)

输入数字：

age = int(input("年龄:"))

5. 运算符

算术运算

a = 10
b = 3

print(a+b)
print(a-b)
print(a*b)
print(a/b)
print(a//b)   # 整除
print(a%b)    # 取余
print(a**b)   # 幂

比较运算

a > b
a < b
a == b
a != b
a >= b
a <= b

逻辑运算

and
or
not

例：

age = 20

if age > 18 and age < 60:
    print("成年人")

6. 条件判断

if

age = 18

if age >= 18:
    print("成年")

if else

age = 15

if age >= 18:
    print("成年")
else:
    print("未成年")

if elif else

score = 90

if score >= 90:
    print("优秀")
elif score >= 60:
    print("及格")
else:
    print("不及格")

7. 循环

while循环

i = 0

while i < 5:
    print(i)
    i += 1

for循环

for i in range(5):
    print(i)

输出：

0
1
2
3
4

遍历列表：

names = ["Tom","Bob","Jack"]

for name in names:
    print(name)

8. range()

range(5)

表示：

0 1 2 3 4

指定范围：

range(1,5)

结果：

1 2 3 4

步长：

range(0,10,2)

结果：

0 2 4 6 8

9. 字符串

创建：

s = "Python"

索引：

print(s[0])

输出：

P

切片：

print(s[0:3])

输出：

Pyt

常用方法：

s.upper()       # 大写
s.lower()       # 小写
s.replace()     # 替换
s.split()       # 分割
len(s)          # 长度

10. 列表 list

创建：

nums = [1,2,3,4]

print(nums[0])

添加：

nums.append(5)

删除：

nums.remove(3)

len(nums)

for n in nums:

11. 元组 tuple

不可修改。

t = (1,2,3)

12. 字典 dict

创建：

person = {
    "name":"Tom",
    "age":20
}

读取：

print(person["name"])

修改：

person["age"] = 21

添加：

person["city"]="LA"

遍历：

for key,value in person.items():
    print(key,value)

13. 集合 set

创建：

s = {1,2,3}

添加：

s.add(4)

删除：

s.remove(2)

特点：

  • 无序
  • 不重复

hello()

16. 匿名函数 lambda

普通：

def add(x,y):
    return x+y

lambda：

add = lambda x,y:x+y

print(add(1,2))

17. 类 Class

    def __init__(self,name):
        self.name=name

23. Python缩进规则

Python 使用缩进表示代码块。

正确：

if True:
    print("yes")

错误：

if True:
print("yes")

推荐：

  • 4个空格缩进
  • 不混用Tab和空格
